┌─ NOTIFICATION SYSTEM ───────────────────────────────────────────────────────┐
│                                                                              │
│  All CRUD operations emit Socket.IO notifications                           │
│                                                                              │
│  📧 Notification Payload                                                   │
│     {                                                                       │
│       type: 'create'|'update'|'delete'|'approval_pending'|'approval_rejected'│
│       title: "User-facing title",                                          │
│       message: "Detailed description",                                     │
│       data: {                                                              │
│         resource_type: 'problem'|'event'|'blog'|'profile',                │
│         resource_id: ID,                                                   │
│         status: 'pending'|'approved'|'rejected',                           │
│         action_url: '#problems'|'#events'|'#blogs'|'#approvals'           │
│       }                                                                     │
│     }                                                                       │
│                                                                              │
│  📨 When Notifications Are Sent                                            │
│  ├─ Problem Created (auto) → Self: "Problem created"                      │
│  ├─ Problem Created (pending) → Self + Admins: "Approval needed"          │
│  ├─ Problem Updated → Self: "Problem updated"                             │
│  ├─ Problem Deleted → Self: "Problem deleted"                             │
│  ├─ Event/Blog (same as Problems)                                         │
│  ├─ Faculty Approves → Admins: "Faculty approved [Item]"                  │
│  └─ Faculty Rejects → Submitter: "Your [Item] was rejected"               │
│                                                                              │
│  🔒 Recipient Selection                                                    │
│  ├─ Self notifications: socket.emit('notification', ...)                  │
│  ├─ Admin notifications: io.emit('notification', ...)                     │
│  └─ Broadcast notifications: socket.broadcast.emit(...)                   │
